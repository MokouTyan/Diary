【2022.06.24】防删库跑路，rm命令的替代者——trash的安装

### 前言

因为前几天新增硬盘，删除软连接的时候unlink失败了，所以想用rm来删除软连接

结果一不小心把/usr/lib目录下的文件都删了，输完命名就觉得大事不妙

只好趁着还没关机，赶紧把所有的docker文件拷出来，损失不至于扩大

所以我打算不再使用rm这个命令，太哈人了

### 安装

安装命令很简单

```
sudo apt install trash-cli
```

### 使用

普通的删除就是trash 文件

要查看已经删除的文件，则使用

```
trash-list
```

### 复原

```
restore-trash
```

选择序号复原

### 清空回收站

```
trash-empty
```
